### AI生成图片
1.变量输入，这其中包括了prompt，和图片的一些参数，具体的变量名称来自图片生成网站的api使用文档；
2.Http请求，挂上图片生成网站的api，此时dify右侧的设置栏里会有一些需要填充的参数（比如post），答案就在网站里使用文档的api；
3.然后是从网站中把生成图片的一坨内容扒出来；
4.代码执行，将url（可以理解为对应图片的链接）从那一坨内容中提取出来；
5.生成图片。
